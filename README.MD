# Face clothes matching

## Настройка окружения

### Установка необходимых пакетов

```bash
sudo apt install tmux htop python-is-python3 pipenv zip
```

### Утсановка python3.7

```bash
sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt install python3.7
```

### Запуск среды

```bash
./run_env.sh
```
